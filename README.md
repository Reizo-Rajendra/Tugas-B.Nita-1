# Biodata Siswa X PPLG 1

Website biodata pribadi untuk tugas mata pelajaran Pemrograman Web.

## Deskripsi

Website ini berisi informasi biodata pribadi siswa kelas X PPLG 1 yang dibuat menggunakan PHP, HTML, CSS, dan JavaScript.

## Fitur

- Biodata lengkap dengan foto profil
- Keinginan 10 tahun ke depan
- Daftar skill dengan progress bar
- Hobi dan kesukaan
- Riwayat pendidikan
- Jadwal pelajaran harian
- Jadwal piket kelas
- Motto hidup yang bergerak

## File

- `Biodata.php` - Halaman biodata utama
- `Jadwal.php` - Halaman jadwal pelajaran dan piket

## Cara Menggunakan

1. Download semua file
2. Letakkan di folder htdocs (jika pakai XAMPP) atau folder web server lainnya
3. Buka browser dan akses `localhost/nama-folder/index.php`

## Teknologi

- PHP
- HTML5
- CSS3
- Font Awesome (untuk icon)
